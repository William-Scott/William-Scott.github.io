微信小程序新音频API

文章

微信小程序基础库升级1.6.0以后有了新的音频调用API接口，对比来说是比之前的接口功能更强，使用更方便

旧版

之前的函数是wx.createAudioContext(audioId, this)和audioContext,调用方法为：

wx.createInnerAudioContext()

创建并返回 audio 上下文 audioContext 对象。在自定义组件下，第二个参数传入组件实例this，以操作组件内 <audio/> 组件

audioContext

通过 audioId 跟一个 <audio/> 组件绑定，通过它可以操作对应的 <audio/> 组件。

方法 参数 说明

seek position 跳转到指定位置，单位s
